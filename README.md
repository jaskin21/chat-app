# chat-app
mernstach with socket.io


# Requirements

- [Node JS](https://nodejs.dev)
- [pnpm](https://pnpm.io)
- [Mongo DB](https://www.mongodb.com) _(Optional)_

---

# Installation

### Client

just run

```sh
pnpm install
```

---

# Run Project

Once the dependencies are installed:

### Client

just run:

```sh
npm run start
```
